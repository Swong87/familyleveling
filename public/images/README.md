# Image Guide

This folder contains all images for the Family Leveling landing page.

## Folder Structure

```
public/images/
├── hero/           # Hero banner image
├── products/       # Individual product images
├── collections/    # Collection category images
└── about/          # About section images (optional)
```

## Required Images

### 1. Hero Banner (`hero/`)
- **File**: `hero-banner.jpg` or `hero-banner.png`
- **Recommended size**: 1920x800px (or similar wide aspect ratio)
- **Description**: Main banner image for the hero section. Should be visually appealing and represent your brand/family gaming theme.
- **Usage**: Displayed at the top of the landing page

### 2. Product Images (`products/`)
You need **4 product images** for the featured products section:

1. **`product-1-cuteness-max.jpg`**
   - Product: Cuteness Level MAX Gamer Girl Hoodie
   - Recommended size: 800x800px (square)
   - Should show the hoodie clearly

2. **`product-2-astronaut.jpg`**
   - Product: Cozy Galaxy Astronaut Hoodie for Kids
   - Recommended size: 800x800px (square)
   - Should show the hoodie clearly

3. **`product-3-christmas-tree.jpg`**
   - Product: Merry Controller Christmas Tree Game Hoodie
   - Recommended size: 800x800px (square)
   - Should show the hoodie clearly

4. **`product-4-reindeer.jpg`**
   - Product: Reindeer Feast Mode Activated Christmas Sweatshirt
   - Recommended size: 800x800px (square)
   - Should show the sweatshirt clearly

### 3. Collection Images (`collections/`)
You need **3 collection images** for the collections section:

1. **`collection-gamer.jpg`**
   - Collection: Gamer Shirts
   - Recommended size: 1200x900px (4:3 aspect ratio)
   - Should represent gaming/family gaming theme

2. **`collection-theme-park.jpg`**
   - Collection: Theme Park Shirts
   - Recommended size: 1200x900px (4:3 aspect ratio)
   - Should represent theme park/adventure theme

3. **`collection-movie.jpg`**
   - Collection: Movie Shirts
   - Recommended size: 1200x900px (4:3 aspect ratio)
   - Should represent movies/cinema theme

### 4. Optional Images (`about/`)
- **Logo/Favicon**: `logo.png` or `favicon.ico` (32x32px or 16x16px)
- **Family image**: Optional image for the about section

## Image Format Guidelines

- **Format**: JPG (for photos) or PNG (for graphics with transparency)
- **Optimization**: Compress images before uploading to reduce file size
- **Naming**: Use lowercase, hyphens for spaces (e.g., `product-1-cuteness-max.jpg`)
- **Quality**: High quality but optimized for web (aim for < 500KB per image)

## After Adding Images

Once you've added your images, the code will automatically reference them. The paths in the code will be:
- Hero: `/images/hero/hero-banner.jpg`
- Products: `/images/products/product-1-cuteness-max.jpg`, etc.
- Collections: `/images/collections/collection-gamer.jpg`, etc.
